Kontaktverlauf

Der Kontaktverlauf ist der rechte Teil der Maske. Jeder Kontakt mit einem Kunden, einem Lieferanten oder einer Person kann hier dokumentiert werden.

Um einen neuen Eintrag anzulegen klickt man einfach auf "Neuer Eintrag". Um einen vorhandenen Beitrag weiter oder wieder zu bearbeiten klickt man einfach auf den Eintrag. Es geht sofort die Eingabemaske für den Kontakt auf. Hier kann im "Betreff" ein solcher eingegeben werden. In dem Feld "Bemerkungen" können die Informationen abgelegt werden. Mit dem Button "Durchsuchen" kann eine Datei oder ein Dokument an den Verlauf angehängt werden. In dem Feld "Dateibeschreibung" gibt man am besten eine gute Beschreibung der Datei ein um auch nach einiger Zeit noch wissen zu können worüber die Datei handelt.

In der ersten Zeile im ersten Pulldownmenü kann der Kontakt für einen verhinderten Kollegen angelegt werden, im zweiten Pulldownmenü kann die Firma oder entsprechende schon angelegte Kontaktpersonen mit diesem Kontakt verknüpft werden, mit dem drittem Pulldownmenü kann der Kontakt zu einem anderen Kontaktverlauf verschoben werden. Der Button "verschieben" löst die Aktion aus.

Zum Schluß kann die Kontaktart angegeben werden und über den Button "neuer Eintrag" wird der Verlauf abgespeichert. Es können mehrere Einträge zu einem Betreff oder Thread gemacht werden. Die Kontaktarten werden farblich wie folgt dargestellt: Telefon T gelb, eMail M grün, Brief/Fax S blau, (schriftlich) Persönlich P lila, Datei D rot, Termin X magenta.


Ist in der Konfiguration "löschen" und "bearbeiten" erlaubt worden, können Eintr&age; verändert und gelöscht werden. Die Originalbeiträge werden aber in einer Historie-Tabelle gesichert. Gibt es hier Einträge, wird ein zusäzlicher Link angezeigt, mit dem man sich die Veränderungen ansehen kann.
