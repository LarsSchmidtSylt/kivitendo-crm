Eingeben und bearbeiten des Projektes
Wenn Sie nicht aus einer Firmenmaske kommen, geben Sie einen einen (Teil-)Name ein und klicken Sie auf die Lupe. Im aufgepopten Fenster wählen Sie die gewünschte Firma aus und bestätigen übernehmen.
In der zweiten Zeile wird eine pregnante Projektbezeichnung eingegeben. Das darauf folgende Feld nimmt eine genauere Berschreibung auf.
Legen Sie das Start- und / oder Stoppdatum fest. Im Feld "Zielvorgabe" geben Sie die veranschlagte Zeit in Stunden ein.
Der Benutzer, der das Projekt angelegt hat, kann mit der Auswahl "Aktiv" das Projekt für weitere Eingaben sperren.
Ein Klick auf den grünen Haken speichert die Eingaben. Mit dem Symbol Haus kommen Sie in die Stammdatenmaske der gewählten Firma. Das kleine quadratische Symbol löscht die Eingabemaske ab. Die Lupe sucht nach Projekten, die den Eingaben entsprechen.

Eingabe der Arbeiten
Wurde ein Projekt ausgewählt, stehen unter dem Projekt weitere Eingabefelder.
Das erste Feld nimmt das Startdatum der Arbeit auf, das zweite Feld die Uhrzeit. Soll die Arbeit jetzt im Moment beginnen, aktivieren Sie statt dessen die erste Checkbox "jetzt". In das grosse Feld geben Sie eine Beschreibung der durchzuführenden Arbeit ein. Mit "sichern" speichern Sie die Eingaben. Die Maske wird abgelöscht und es wird eine neue Zeile unter der Eingabezeile erstellt.
Ein Klick auf die/eine Zeile öffnet die hinterlegten Daten wieder in der Eingabemaske. Die Daten können nun ergänzt werden. Eben so wie der Start der Arbeit kann das Ende in den beiden Feldern "Arbeitsende" eingegeben werden. Speichern Sie wieder Ihre Eingaben.
Ein offener Arbeitseintrag wird durch den Text "jetzt beenden" nach dem Startdatum gekennzeichnet. Ein Klick darauf schließt den Arbeitseintrag zu diesem Zeitpunkt.
Immer wenn ein Arbeitseintrag beendet wird, wird die verbrauchte Zeit aufgerechnet und vom vorgegeben Zeitrahmen abgezogen. Beide Werde werden ausgegeben.

Abrechnen der Arbeiten
Um einzelne Posten abzurechnen, markieren Sie die einselnen Zeilen durch anklicken der Checkboxen. Um alle Zeilen abzurechnen markieren Sie die Box vor "alle".
Klicken Sie nun auf "abrechnen" um die Abrechnung zu starten. Bei abgerechneten Zeilen wird die Checkbox durch ein Minuszeichen ersetzt.
Unter Verkauf -> Berichte -> Aufträge finden Sie nun einen neuen Auftrag für den Kunden über die markierten Zeilen. Die Arbeitszeiten werden aufgerundet. Die tatsächliche Zeit in der Seriennummer gespeichert.
Um eine abgerechnete Zeile noch einmal abrechnen zu können, muss die Zeile zum Bearbeiten geladen werden. Ändern Sie eine der Zeiten und speichern Sie dann den Arbeitseinsatz. Der original Auftrag bleibt aber erhalten.

Abrechnen der Arbeiten
Geben Sie eine Suchbegriff in eine der ersten drei Felder ein und klicken Sie auf das Lupensymbol. Wird nur ein passender Auftrag gefunden, so wird dieser gleich geladen.
Sind mehrere Treffer da, so werden diese in einer Auswahlbox dargestellt. Wählen Sie nun einen aus und klicken Sie auf ok.
