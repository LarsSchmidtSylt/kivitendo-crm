DetailansichtLieferantendaten

Hier werden die Lieferantendaten detailiert angezeigt. Die Detailansicht wird über vier Reiter navigiert. Der aktive''' Reiter hat eine '''graue Schrift

[Lieferantendaten]

Das dreigeteilte Anzeigefenster enthält Informationen über die Rechnungsanschrift, abweichende Lieferadresse und einen [Kontaktverlauf].

=[Rechnungsanschrift]=

In dem Fenster der Rechnungsanschrift wird mit einem Klick auf das Briefsymbol ein neues Fenster geöffnet. Dort wird die Adresse als reiner Text dargestellt um sie mit Copy + Paste in eine andere Anwendung zu übernehmen. Es wird auch eine Möglichkeit Etiketten zu drucken angeboten. Unterhalb des Briefsymbols werden die Kundesnummer des Lieferanten im System und darunter die Kundennummer beim Lieferanten dargestellt. Ist eine eMail-Adresse eingegeben, so wird diese als Link dargestellt und führt zu dem eingebauten eMail-Client. Eine evtl. vorhandene Web-Adresse wird auch als Link angezeigt und öffnet ein neues Fenster mit der angegebenen URL.

=[Lieferadresse]=

In dem Fenster "Lieferadresse" wird die möglicherweise abweichende Lieferadresse von der Rechnungsanschrift mit den eventuell verschiedenen Telefon, Fax und als Link dargestellter eMail-Adresse dargestellt. Das Briefsymbol erzeugt ein Popup mit der als Text dargestellten Adresse um sie wieder übernehmen zu können und die Möglichkeit Etiketten der Lieferadresse drucken zu lassen.

===Kontaktverlauf ===

Im rechten Teil der Anzeige werden, wenn vorhanden, die Kontaktereignisse aufgeführt und es wird die Möglichkeit geboten einen neuen [Verlauf] einzustellen. Ein Klick auf die entsprechende Zeile öffnet einen entsprechenden Dialog.

Unterhalb der Adressen führt der link "Edit" in die Maske [Lieferanten eingeben/editieren]. Die Daten können dort verändert werden.

[Kontakte]

Wird dieser Reiter angeklickt, werden zunächst alle Kontaktpersonen der Firma angezeigt. Ein Klick auf die gewünschte Person stellt dann diese ein. Sollte es zu der Firma nur einen Kontakt geben, wird dieser automatisch angezeigt. Das zweiteilige Anzeigefenster enthält im linken Teil die Daten einer Kontaktperson, im rechten Teil erscheint wieder ein [Kontaktverlauf] mit dieser Person. Unterhalb der Anzeige gibt es wieder die Möglichkeit die Kontaktdaten zu bearbeiten oder einen Kontakt aus der [Personensuche] zu wählen.

== Umsätze ==

Ein erster Klick auf den Reiter "Umsätze" stellt die summierten Umsatzdaten der letzten 12 Monate als tabellarische Liste und als Grafik dar. Die Grafik stellt die Umsatzdaten logarithmisch dar, mit einem weiteren Klick auf den Reiter "Umsätze" kann in eine lineare Darstellung umgeschaltet werden. Dargestellt werden je Monat: - Gesamtumsatz - durchschnittlicher Umsatz - Gutschriften Mit einem Klick auf einen Tabelleneintrag, wird in die Monatsansicht umgeschaltet. Jetzt sind alle Rechnungen und Angebote einzeln sichtbar. Die einzelnen Rechnungspositionen können in einem neuen Fensten durch anklicken dargestellt werden.

[Dokumente]

Hier kann eine Dokumentvorlage ausgewählt werden. [Dokumentvorlagen] können zur Zeit im OpenOffice- und im RTF-Format bearbeitet werden. Die Dokumentvorlagen müssen vor Ihrer eigentlichen Nutzung einmalig angelegt und vorkonfiguriert werden. Mit einem Klick auf das gewünschte Dokument kommen Sie in die [Variablenansicht]. Verschiedene Felder im Dokument werden mit diesen Variablen ersetzt. Dabei können verschiedene Felder bereits mit den Daten eines Lieferanten oder Kontaktperson ausgefüllt sein. Die Felder für Datum und Zeit beziehen sich auf die Serverzeit. Sind alle Felder zu Ihrer Zufriedenheit ausgefüllt drücken Sie einfach nur "Erstellen". Damit wird das eigentliche Dokument erzeugt und zum Speichern angeboten. Gleichzeit wird das Dokument auch bei der entsprechenden Adresse im [Kontaktverlauf] zugeordnet.
